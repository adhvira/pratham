# Grants

Funding opportunities for first-time Indian founders — from government schemes, institutions, and private foundations.

## Files

| File | What it covers |
|---|---|
| `government.md` | Central and state government schemes |
| `institutions.md` | University, IIT, IIM, and institutional grants |
| `private.md` | Private foundation and corporate grants |

## How to contribute

Add a grant to the relevant file. Use this format:

```markdown
### Grant / Scheme Name
**Source:** Organisation name  
**Amount:** ₹X – ₹Y  
**Eligibility:** Who can apply  
**Stage:** Idea / Pre-revenue / Early revenue  
**Deadline:** Month Year / Rolling / Annual  
**Apply:** [Link](url)  
**Notes:** Any important conditions or tips  
**Last verified:** Month Year  
```

## Important

Deadlines change. Links go dead. If you notice an outdated entry, open a PR to fix it or raise an issue. The value of this directory depends on it being accurate.
