# Learn

Stage-wise knowledge for first-time founders. Every resource here has been contributed by someone who has walked this path.

## Stages

| File | Stage | What it covers |
|---|---|---|
| `pre-idea.md` | Before the idea | Finding problems worth solving |
| `idea-stage.md` | You have an idea | Sharpening it, stress-testing it |
| `validation.md` | Testing the idea | Talking to users, finding signal |
| `building.md` | Building the product | Shipping, iterating, not overthinking |
| `scaling.md` | Growing beyond early traction | Distribution, team, money |

## How to contribute

Add a resource to the relevant stage file. Use this format:

```markdown
### [Resource Title](link)
**Type:** Book / Video / Podcast / Case Study  
**Author / Channel:** Name  
**Why it helps:** One sentence on what a founder gets from this.
```

Only add resources you have personally used or can genuinely recommend.
