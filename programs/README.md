# Programs

City-wise directory of incubators, accelerators, and programs supporting early-stage Indian startups — free or at nominal cost.

## Files

| File | Coverage |
|---|---|
| `hyderabad.md` | Hyderabad and Telangana |
| `bangalore.md` | Bangalore and Karnataka |
| `mumbai.md` | Mumbai and Maharashtra |
| `delhi.md` | Delhi NCR |
| `national.md` | Pan-India programs |

## How to contribute

Add a program to the relevant city file. Use this format:

```markdown
### Program Name
**Organisation:** Name  
**City:** City / Online  
**Stage:** Idea / Pre-revenue / Early revenue  
**Cost:** Free / Nominal (₹X) / Paid  
**Duration:** X months  
**What they offer:** 2–3 lines  
**Apply:** [Link](url)  
**Deadline:** Month Year / Rolling  
**Last verified:** Month Year  
```

Only list programs that are currently active. If a program closes, open a PR to remove or mark it inactive.
